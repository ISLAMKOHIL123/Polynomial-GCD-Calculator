# Polynomial GCD Calculator

An interactive web app for computing the GCD of polynomials and modular inverses
in finite fields Fₚ[x], with step-by-step algorithm visualization.

🔗 **Live demo**: https://tparmo2.vercel.app

## What it does

- Computes **GCD of polynomials** over a prime field Fₚ using the Euclidean algorithm
- Computes **modular inverse** of a polynomial using the Extended Euclidean algorithm
- Shows every step of the algorithm in real time
- Supports arbitrary polynomials with coefficients mod p

## Features

- Set any prime modulus p
- Enter polynomial coefficients degree by degree
- Switch between GCD and Modular Inverse modes
- Step-by-step breakdown of each division and reduction

## Tech Stack

Pure HTML5 + CSS3 + JavaScript — no build step, no dependencies to install.
Uses Bootstrap 5 and Font Awesome via CDN.

## Usage

```bash
open index.html
```

1. Set your prime p
2. Choose operation (GCD or Modular Inverse)
3. Enter polynomial coefficients
4. Click Compute and follow the steps
